Поддержка KSkype на базе Skype for linux

KSkype это программа для аудио/видео звонков, обмена мгновенными сообщениями по протоколу SkypeWebAPI. Основой для версий 1.41-1.43х использовался Скайп для Линукс (Сборка ReSkype 2.5, адаптированная под Windows человеком под ником V1od). Главная задача KSkype - повторить максимально интерфейс классического Скайп 7.41
