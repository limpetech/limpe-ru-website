Аутентификация
Каждый запрос в API Твоего Мира сопровождается куки с именем PHPSESSID, она является своеобразным токеном аутентификации. Для получения PHPSESSID и старта сессии необходимо отправить запрос на https://my.dexpaz.ru/api/legacy/v1/login.php методом POST с параметрами login и password, где login - логин пользователя, а password - пароль. Если аутентификация завершилась успешно, сервер отправит ответ в формате JSON и куки PHPSESSID:
  
{
"code": 200,
"message": "Успех!", 
"data": { "id": 1 } 
} 

Получение данных о пользователях 
С помощью API вы можете получить данные о пользователях, для этого необходимо отправить GET-запрос на https://my.dexpaz.ru/api/legacy/v1/getUser.php с параметром id, которому присвойте айди пользователя, чью информацию хотите получить. Если запрос удачен, вы получите ответ в формате JSON. Пример: 
{ 
"code": 200, 
"message": "Успех!", 
"data": { "name": "Time", 
"surname": "Inception", 
"gender": "1", 
"verify": "5", 
"nickname": "", 
"aboutuser": "", 
"birthdate": "01.1.1970", 
"lastonlinedate": "", 
"lastonlinetime": "", 
"status": "Прывэт", 
"avatar": "content/avatars/1061391416.jpg" 
} 
 } 

Получение писем/сообщений 
Твой Мир использует формат сообщений OpenVK, который в Твоём Мире называется Почтой@Твой Мир. Для того, чтобы получить входящие или исходящие сообщений, отправьте GET-запрос на https://my.dexpaz.ru/api/legacy/v1/getMessages.php с куки PHPSESSID и параметром mailbox, который отвечает за ящик, из которого необходимо получить письмо. inbox - для входящих, outbox - для исходящих. Если запрос удачен, вы получите ответ в формате JSON: 
{ 
"code": 200, 
"message": "Успех!",
"data": [ { "id": "67", "to": "1", "topic": "Ура", "text": "робе" }, 
{ "id": "54", "to": "7", "topic": "ого", "text": "ого" }, 
{ "id": "50", "to": "2", "topic": "ку", "text": "ку" } ] 
} 

Отправка писем/сообщений 
         Важно! В Твоём Мире можно отправлять сообщения только друзьям. Не пытайтесь          отправить сообщения незнакомцам, сервер ответит вам кодом 403. 

Для отправки писем в Твоём Мире существует конечная точка sendMessage с URL https://my.dexpaz.ru/api/legacy/v1/sendMessage.php. Пошлите на неё POST-запрос с куки PHPSESSID и параметрами id, в который необходимо указать айди адресата, topic, в которую необходимо указать тему письма и text, где необходимо указать само содержание. В случае успеха сервер пришлёт ответ с кодом 200.
